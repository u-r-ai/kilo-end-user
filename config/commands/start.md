# /start — Mulai Project Baru

Gunakan command ini untuk memulai project baru dari awal.

## Yang Dilakukan

1. Tanyakan nama project ke user
2. Tanyakan jenis project (web app, API, automation, landing page, dll)
3. Buat folder project di direktori saat ini
4. Buat struktur folder standar
5. Buat file awal (README, .env.example, .gitignore, Dockerfile)
6. Inisialisasi git repository
7. Jalankan setup awal sesuai jenis project

## Struktur Default

```
nama-project/
├── README.md
├── .gitignore
├── .env.example
├── Dockerfile
├── docker-compose.yml
├── src/
├── tests/
└── docs/
```

## Output

Setelah selesai, tampilkan:
- Lokasi project
- Struktur folder yang dibuat
- Langkah selanjutnya untuk user
