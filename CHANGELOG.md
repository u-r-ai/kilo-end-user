# Changelog

Semua perubahan signifikan pada proyek ini akan dicatat di sini.

Format mengikuti [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] — 2025-01-15

### Added
- Rilis perdana Kilo End-User
- Installasi satu command via `curl | bash`
- Agen AI berbahasa Indonesia dengan mode primary
- Dukungan provider: DeepSeek, Anthropic, OpenAI, Google Gemini, OpenRouter
- Integrasi MCP: filesystem, git, docker, memory
- Command `/start` untuk memulai project baru
- Command `/status` untuk mengecek kondisi sistem
- Project Builder skill dengan panduan lengkap
- Docker Compose otomatis untuk setiap project
- Panduan best practice di `docs/best-practices.md`
- Dukungan install di Linux dan Windows (WSL)
